Admin UI removed

The in-browser admin UI has been removed. User data is now stored in MongoDB so you can manage users directly with MongoDB Compass or via the server API.

  • Connect to your MongoDB instance and open the users collection (database used by this app) to view/edit accounts.
  • To add an admin manually, insert a document with fields: username, password (bcrypt hash), and role: "admin".
  • To log in via the site, use the credentials stored in the DB and then access pages as usual.

If you need, I can add a small management API or a secure admin UI later.
